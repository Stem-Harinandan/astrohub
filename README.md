AstroHub is a free, open-source desktop app for astrophotography processing. It’s built to solve a simple problem: existing tools are either expensive, outdated, or locked to one platform. Software like PixInsight is powerful but costly, while DeepSkyStacker is limited and not cross-platform.

AstroHub runs entirely locally, so there are no uploads or file size limits. You can load your light frames, add darks and flats if you have them, and run a full pipeline calibration, alignment, and stacking in one go.

The focus is on making the process simple and fast. In a few minutes of interaction, you should be able to turn raw frames into a clean, usable image without fighting the software.

It’s built with Python using NumPy, Astropy, OpenCV, and PyQt, keeping it lightweight but powerful. The UI is minimal and modern, designed to stay out of your way.

Right now, it’s about getting the basics right. Everything else comes later.
